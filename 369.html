369 Game / 369 게임

게임이 끝난 숫자 N이 주어졌을 때, N 이전까지 박수를 친 횟수를 구하여라.

Find how many times you have clapped right before number 'N'.

예시 / Example
--------------

35가 주어지면, 박수를 친 횟수가 15가 된다. 3, 6, 9, 13, 16, 19, 23, 26, 29, 30, 31, 32, 33, 33(3이 두개니까 두번), 34

In this game, the first person yell out one! then person next to that person yell two! and then we keep on counting the numbers until one people make mistake. But, there is some variation. Everytime a person have to count 3, 6, 9, and other numbers that have 3, 6, or 9 in, the person that is suppose to count that number have to clap. In this program, If input is 35, you have clapped 15 times before game ends. 3, 6, 9, 13, 16, 19, 23, 26, 29, 30, 31, 32, 33, 33(twice, because there is two threes in 33), 34

inp = input()
clap = 0
all_num = []
for num in range(int(inp) - 1):
    all_num.append(num + 1)
for let in all_num:
    letter = str(let)
    if "3" in letter or "6" in letter or "9" in letter:
        for i in range(len(letter)):
            if int(letter[i]) == 3 or int(letter[i]) == 6 or int(letter[i]) == 9:
                clap += 1
print(clap)
